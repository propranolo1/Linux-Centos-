# 进程指南

## 进程介绍

每执行一个程序都会有一个进程

而每一个进程都会对应一个父进程

前台 目前看得到的

后台 看不见的

## 查看进程（ps）

```
ps -a *显示当前终端所有的进程信息
ps -u *以用户的格式显示进程信息
ps -x *显示后台进程运行的参数
ps -ef *查看程序的父进程
pstree *树状图查看进程
```

## 终止进程

将进程进行终止，支持通配符

-9为强制终止的含义

```
kill 进程PID *终止该PID的进程
killall 进程名称 *终止所有该名称的进程
kill -9 进程pid *强制终止该PID进程
```
